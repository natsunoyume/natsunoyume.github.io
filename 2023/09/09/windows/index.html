Microsoft Windows

Windows 是微软以图形用户界面为主推出的一系列专有商业软件操作系统

  • 版本
    • 早期版本（Windows NT 3.1/3.5/3.51/4.0/2000）
    • Windows XP
    • Windows Vista
    • Windows 7
    • Windows 8 / 8.1
    • Windows 10
    • Windows 11

用户帐户

Windows用户有帐户名、密码、用户安全标识（SID）

注：SID安全标识符（身份证号），每个用户的SID唯一

打开本地用户和组运行 - lusrmgr.msc

内置帐户

  • administrator:管理员，权限最大
  • guest:来宾，权限小，默认禁用

    注：内置帐户可以改名、禁用，不能删除

常见内置组及权限

  • Administrators：管理员组
  • Guests:来宾组
  • Network Configuration Operators：具有管理网络功能，例如更改IP地址
  • Backup Operators：具有备份和还原的权限
  • Print Operators：具有管理打印机的权限
  • Remote Desktop Users：此组内用户可以从远程计算机使用远程桌面来连接
  • Users:新用户默认组
  • Everyone: 任何用户都属于此组
  • Authenticated users:任何有效登录的用户都属于此组

NTFS

NTFS（New Technology File System）是微软开发的一种文件格式，专门为网络和磁盘配额、文件加密等管理安全特性设计

常见文件系统类型

  • FAT32
  • ext4
  • NTFS

特点

  • 高读写
  • 磁盘空间利用率高（支持压缩、磁盘配额）
  • 安全性高（支持加密、NTFS权限配置）
  • AD需要NTFS支持

NTFS权限

  • 完全控制：修改文件，给其他用户分配权限
  • 修改：读、写、删、打开程序
  • 读取和执行：读、打开程序
  • 列出文件内容：读取文件夹内容
  • 读：读取内容
  • 写：写入内容
  • 特殊权限：

NTFS权限的规则

权限累加

  1. Administrator用户创建02.txt
  2. 添加普通用户-勾选读取
  3. 添加用户所属组-仅勾选写入
  4. 普通用户登录打开02.txt，能更改文件内容

注：用户与用户所属组权限不冲突，权限累加

  1. Administrator用户创建03.txt

  2. 添加普通用户勾选读取

  3. 添加用户所属组拒绝读取

  4. 普通用户登录打开03.txt,拒绝访问

  5. Administrator登录

  6. 添加普通用户-勾选拒绝读取

  7. 添加用户所属组允许读取

  8. 普通用户登录打开03.txt,拒绝访问

权限的拒绝（拒绝大于一切、拒绝优先）

权限的继承

继承：默认下级继承上级目录的权限

  1. Administrator用户创建文件夹：E:\int
  2. 右击int 目录-属性-安全-编辑-添加普通用户完全控制权限
  3. 在int的目录中创建01.txt、02.txt、03.txt
  4. 右击01.txt – 属性-安全-查看是否有普通用户完全控制权限
取消继承
  1. 右击01.txt-属性-安全-高级-禁用继承
  2. 在弹出的提示中选择第一个-确定
  3. 编辑-选择普通用户-取消完全控制的勾选仅保留读写权限-确定
强制继承
  1. 右击int-属性-安全-高级
  2. 勾选禁用继承下的小方块-是-确定

    注：下级文件取消继承后上级目录可以强制继承，上级目录强制继承后下级文件还可以取消继承

文件共享

  • 路径访问共享
  • 映射网络驱动器访问共享

磁盘

  • 类型
    • 基本磁盘：传统默认磁盘，只能管理本磁盘的分区
    • 动态磁盘：较强的可扩展性、高读写、高可靠
  • 分区形式：MBR，GPT
    • 打开磁盘管理器
    • 运行- diskmgmt.msc

动态磁盘（软RAID）

  • 简单卷
    • 仅需一块动态磁盘即可创建简单卷，将基本磁盘转换为动态磁盘
  • 跨区卷
    • 至少两块动态磁盘
    • 成员空间大小可不同
    • 容量是组成跨区卷的成员空间总和
    • 可扩展
  • 带区卷
    • 至少两块动态磁盘
    • 成员空间大小相同
    • 容量是组成带区卷的所有成员空间总和
    • 读写速度快
    • 以带区形式在两个或多个磁盘上数据被均匀以带区形式跨磁盘交替分配
  • 镜像卷
    • 只能两块动态磁盘
    • 成员空间大小相同
    • 空间利用率50%
    • 支持容错（故障转移），增加可靠性
    • 一般系统存在镜像卷
  • 创建与修复镜像卷
    • 创建镜像卷并存入内容
    • 关机移除镜像卷成员之一，再添加一块新硬盘
    • 启动计算机-打开磁盘管理器
    • 初始化磁盘并转换为动态磁盘
    • 右击之前镜像像盘符-选择删除镜像-选择丢失-删除镜像-是
    • 右击曾经失败的镜像卷（此时是普通的简单卷）-添加镜像
  • RAID5卷
    • 至少三块动态磁盘
    • 成员空间大小相同
    • 空间利用率(n-1)/n
  • 创建与修复RAID5卷：
    • 通过三块动态磁盘创建RAID5卷并存入内容
    • 关机移除RAID5卷成员之一，再添加一块新硬盘
    • 启动计算机-打开磁盘管理器
    • 初始化磁盘并转换为动态磁盘
    • 右击失败的RAID5卷-修复卷
